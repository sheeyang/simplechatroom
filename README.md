## Usage

### Add your firebase config in App.js
```
const  firebaseConfig  =  {
	// your config
}
```

### Install dependencies
```
npm install
```

### Run React dev server (http://localhost:3000)
```
npm start
```

### Build for production
```
npm run build
```
